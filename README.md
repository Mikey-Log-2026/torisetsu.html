# 🗝️ MiKey Log System
### 〜 心を込めた鍵管理システム 〜

Raspberry Pi Picoを使用した、4つの鍵を見守るスマートキーボックスです。  
鍵の持ち出し時間を監視し、LEDの優しい光で状態を教えてくれます。

## 🌟 特徴
- **🔒 自動施錠機能**: PCからの指令で解錠し、10秒後に自動ロック。
- **🚨 半ドア警告**: 扉が開いたままだと、赤いLEDが心配そうに点滅します。
- **⏳ お出かけタイマー**: 1時間を過ぎると、光の色を変えてお知らせします。

## 🔌 配線図（回路のつながり）

| 部品 | Picoピン番号 | 役割 |
| :--- | :--- | :--- |
| **Relay & Green LED** | GP19 | ドアの解錠と状態表示 |
| **Door Red LED** | GP10 | 半ドア警告 |
| **Door Switch** | GP9 | 扉の開閉検知 |
| **Hook 1-4** | (各設定ピン) | 鍵の持ち出し状態を監視 |

## 🚀 使い方
1. `main.py` をPicoに書き込みます。
2. Thonnyなどで `"UNLOCK"` と送信してください。
3. 魔法のように扉が開き、10秒経つと自動で鍵がかかります。
