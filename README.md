# 🗝️ MiKey Log System
### 〜 心を込めた鍵管理システム 〜

Raspberry Pi Picoを使用した、4つの鍵を見守るスマートキーボックスのシステムです。  
鍵の持ち出し時間を監視し、LEDの優しい光で状態を教えてくれます。

---

## 🌟 特徴
* **🔒 自動施錠機能**: PCからの指令で解錠し、10秒後に自動ロック。
* **🚨 半ドア警告**: 扉が開いたままだと、赤いLEDが心配そうに点滅します。
* **⏳ お出かけタイマー**: 1時間を過ぎると、光の色を変えてお知らせします。

## 🔌 配線図（回路のつながり）

| 部品 | Picoピン番号 | 役割 |
| :--- | :--- | :--- |
| **Relay & Green LED** | GP19 | ドアの解錠と状態表示 |
| **Door Red LED** | GP10 | 半ドア警告 |
| **Door Switch** | GP9 | 扉の開閉検知 |
| **Hook 1-4** | (各設定ピン) | 鍵の持ち出し状態を監視 |
## 🌈 動作イメージ
> 鍵が戻ると緑色の光が優しく迎えてくれ、長すぎるお出かけには赤い光が「そろそろ帰ってきてね」と教えてくれます。



## 📝 製作のポイント
* **リミットスイッチの調整**: 鍵の重みでカチッと鳴る位置に調整するのがコツです。
* **配線の色分け**: 4つのフックがあるので、線が混ざらないように色を変えるとメンテナンスが楽になります。

## 🚀 使い方
1.  **書き込み**: `main.py` をRaspberry Pi Picoに保存します。
2.  **解錠命令**: PCのシリアル通信から `"UNLOCK"` と送信。
3.  **動作**: 扉が開き、10秒経つとカチッと自動で施錠されます。
